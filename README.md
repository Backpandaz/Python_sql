# Google-colab-SQLITE-e-Python-
O google colab é conhecido por ser um ambiente on-line que permite o usuário escrever códigos em Python sem a necessidade de qualquer tipo de instação. Mas é possível também fazer consultas utilizando SQLITE3, e a vantagem disso é conseguir integrar bibliotecas do Python e sql; No código "Python+SQLite" você vê a estrutura para conseguir importar as bilblitocas, criar a conexão, criar o database e, por fim, alocar a sua tabela localmente para rodar as consultas em SQL.
